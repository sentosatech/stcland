# CheckboxGroup Component

The `CheckboxGroup` component allows users to toggle a checked and unchecked state with optional icon customization per checkbox. 

### Properties

#### CheckboxOption Base:
- **value** (`boolean`): Unique identifier for the checkbox.
- **checked** (`boolean`): Controls whether the checkbox is checked. Defaults to `false`.
- **onChange** (`() => void`): Callback function that triggers when the checkbox is toggled.
- **label** (`string`): The text displayed alongside the checkbox.
- **indeterminate** (`boolean`): Controls wheter it is in a partial state, mostly used when there is a list of items. Defaults to `false`.
- **disabled** (`boolean`): Disables the checkbox, preventing interaction. Defaults to `false`.
- **primary** (`boolean`): Applies primary styling to the checkbox.
- **neutral** (`boolean`): Applies neutral styling to the checkbox.
- **secondary** (`boolean`): Applies secondary styling to the checkbox.
- **sm** (`boolean`): Applies small size variant. Defaults to `true`.
- **md** (`boolean`): Applies medium size variant.
- **lg** (`boolean`): Applies large size variant.
- **icon** (`React.ReactNode`): Icon displayed in the unchecked state. Requires `checkedIcon` to be passed if used.
- **checkedIcon** (`React.ReactNode`): Icon displayed in the checked state. Requires `icon` to be passed if used.
- **indeterminateIcon** (`React.ReactNode`): Icon displayed in the indeterminate state.

#### CheckboxGroup:
- **options** (`CheckboxOption[]`): Array of CheckboxOptions
- **onChange** (`(updatedOptions: CheckboxOption[])=> void`): Callback for parent to handle state changes.
- **orientation** (`vertical | horizontal`): Layout for checkboxes group.

#### Custom Styles:
- **className** (`string`): Additional CSS classes for the button.
- **customStyles** (`Partial<CheckboxGroupStyles>`): Allows passing custom styles for `root`, `vertical`, `horizontal` and `checkbox`, with various customizations.
